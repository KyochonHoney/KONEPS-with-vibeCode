# 나라장터 입찰 공고 자동 분석 및 제안서 초안 생성 프로젝트

## 🎯 프로젝트 개요
나라장터 입찰 용역 공고를 API를 통해 수집하고, 첨부파일을 다운로드하여 HuggingFace 모델로 문서 분석 및 제안서 초안을 생성합니다. 최종 결과물은 분석안과 제안서 목록 형태의 한글 파일(.hwp)로 자동 생성하는 것을 목표로 합니다.
